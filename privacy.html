O que é o Rachaí

O Rachaí é um aplicativo gratuito para dividir contas em grupo (despesas, lista de compras e acerto de quem paga quem). Esta política explica quais dados o app trata e como.

Dados que você fornece

Ao usar um evento, você cadastra informações como: nome do evento, nomes das pessoas participantes, despesas (descrição e valor), itens de compra e pagamentos. Esses dados ficam vinculados ao evento e são visíveis para quem tem o link do evento. Recomendamos usar apenas o primeiro nome das pessoas e não inserir dados sensíveis.

Integração com IA (API e GPT)

O Rachaí oferece uma API por evento para que você, opcionalmente, use uma IA (como o ChatGPT) para cadastrar itens. Quando você usa essa integração, o texto que você envia à IA e os dados resultantes (nomes, valores, itens) são enviados à API do Rachaí para registrar no seu evento. Cada evento tem um token próprio, que funciona como senha e pode ser revogado. Não coletamos dados além do que você informa para registrar no evento.

Dados de uso (analytics)

Coletamos métricas de uso de forma anônima e agregada: número de visitas, ações realizadas (ex.: criar evento, copiar resumo) e um identificador anônimo de sessão gerado no seu navegador. Não coletamos endereço IP, nome, e-mail ou qualquer dado pessoal nessas métricas, e não usamos cookies de rastreamento.

Onde os dados ficam

Os dados são armazenados no Supabase (infraestrutura de banco de dados em nuvem). O Rachaí não vende nem compartilha seus dados com terceiros para fins de publicidade.

Exclusão de dados

Você pode remover despesas, pessoas, itens e pagamentos diretamente no app. Para solicitar a exclusão completa de um evento, entre em contato pelo e-mail abaixo informando o link do evento.
